# FitAura (Project of 2)

## Overview
FitAura is an AI-driven personalized outfit recommendation system designed to help users discover clothing styles that match their preferences, occasion, mood, weather, and fashion needs through an interactive chatbot experience. The project combines Natural Language Processing (NLP) for understanding user input and intelligent conversation with Generative Adversarial Networks (GANs) for generating fashion-oriented visual recommendations, making the styling process more engaging, smart, and personalized.

## Project Timeline
- **Creation Date**: November 2025
- **Refinements**: December 2025, May 2026

## Key Features
- **Personalized Outfit Recommendations**  
  Suggests outfits based on user preferences, occasion, weather conditions, style choices, and mood.

- **Enhanced Chatbot Interface**  
  Allows users to interact naturally with the system and receive fashion recommendations in a conversational way.

- **AI-Powered Understanding**  
  Uses NLP techniques to analyze user input and understand intent more accurately.

- **Visual Outfit Generation**  
  Integrates GAN-based generation to create fashion-oriented outfit visuals based on user needs.

- **Context-Aware Suggestions**  
  Considers multiple real-life factors such as body attributes, emotional tone, and event type for better recommendations.

- **Iterative Refinement**  
  Users can choose the modify outfit option to refine results and get improved outfit suggestions.

- **Smart Fashion Assistance**  
  Acts like a virtual stylist by guiding users toward more suitable and appealing outfit combinations.

## How to use
1. **Set up the project environment**  
   Clone or download the project files to your local machine. Install the required dependencies and make sure the development environment is ready.

2. **Run the backend server**  
   Start the backend services that handle chatbot responses, recommendation logic, and communication with the database.

3. **Launch the application interface**  
   Open the web or application interface where the user can interact with the chatbot and submit outfit-related queries.

4. **Create or access a user profile**  
   Enter basic user information or preferences such as favorite colors, style type, occasion, mood, or weather context if required by the system.

5. **Interact with the chatbot**  
   Type a request such as:
   - “Suggest an outfit for a formal event”
   - “What should I wear for a beach trip?”
   - “Recommend something casual for rainy weather”

6. **Let the AI process the request**  
   The NLP module analyzes the text, extracts user intent, and maps the input to suitable fashion recommendations.

7. **View the generated recommendation**  
   The system presents outfit suggestions and generates visual outputs based on the selected 11 preferences.

8. **Refine the results**  
   If the recommendation is not perfect, continue the conversation by choosing the modify outfit option.

9. **Explore improved outputs**  
   The system updates the recommendations using feedback and provides more personalized results in the next iteration.

10. **Use the system as a virtual styling assistant**  
    Continue using FitAura for daily outfit planning, special events, seasonal styling, or fashion inspiration.
